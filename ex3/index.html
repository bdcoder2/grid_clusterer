Grid Clusterer - Enabling and disabling cluster marker click checks
Choose setting:
- then click the cluster marker
Note that the map does not "zoom-in" when the cluster marker is clicked, even though internally the clusterer calls the Google maps .fitbounds() method. Because the bounds of the data points (shown in the tile to illustrate), already fit in the map no change occurs.
- then click the cluster marker
When "cluster marker click checks" are enabled, the clusterer will check if the tile containing the cluster marker that was clicked is still visible and if so, will "zoom-in" one level.
